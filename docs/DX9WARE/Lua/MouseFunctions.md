# Mouse Functions

## isLeftClick

**Syntax**

```go
isLeftClick()
```

**Returns**

- a boolean if left mouse button has been clicked.

---

## isRightClick

**Syntax**

```go
isRightClick()
```

**Returns**

- a boolean if right mouse button has been clicked.

---

## isLeftClickHeld

**Syntax**

```go
isLeftClickHeld()
```

**Returns**

- a boolean if left mouse button is being held.

---

## isRightClickHeld

**Syntax**

```go
isRightClickHeld()
```

**Returns**

- a boolean if right mouse button is being held.

---

## MouseMove

**Syntax**

```go
MouseMove({x,y})
```

**Parameters**

- `<table>` table with two values, x & y.
